# Worker-Progress-Report

##  Project Overview
This project is a dynamic **Worker Progress Report** built using **HTML, CSS, and JavaScript**. It generates a PDF-style report similar to the provided sample documents with dynamic data, multiple pages, page numbers, header, footer, and print support.

##  Features
- Dynamic report generation using JavaScript
- Two sample datasets (Short Example & Long Example)
- Responsive A4 page layout
- Header and footer on every page
- Automatic page numbering
- Interactive checkboxes
- Pain scale section
- Print / Save as PDF support

##  Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

##  Project Structure

```
Worker-Progress-Report/
│── index.html
│── style.css
│── script.js
│── logo.jpeg
│── README.md
```

##  How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Use:
   - **Short Example** to load sample data.
   - **Long Example** to load detailed data.
   - **Print / Save PDF** to export the report.
